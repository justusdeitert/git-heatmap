# git-dashboard

A beautiful, live HTML dashboard for any git repository's commit history — styled like GitHub's contribution graph.

![Dark mode](https://img.shields.io/badge/dark%20mode-supported-333?style=flat-square) ![Light mode](https://img.shields.io/badge/light%20mode-supported-eee?style=flat-square&labelColor=666)

## Features

- GitHub-style commit heatmap (last 12 months)
- Stats: total commits, active days, current/longest streak, contributors, busiest day
- Dark/light mode (follows system preference)
- Tooltips on hover
- Live local server — refreshes git data on every page load
- Auto-opens browser, stops on Ctrl+C
- Zero dependencies (uses `node:http`)

## Usage

Run inside any git repository:

```bash
node /path/to/tools/git-dashboard/dist/index.js
```

This starts a local server at `http://127.0.0.1:3333` and opens your browser. Refresh the page to see the latest data. Press Ctrl+C to stop.

Override the port with `PORT=4000 git-dashboard`.

### Global install via npm link

```bash
cd tools/git-dashboard
npm link
```

Then from any repo:

```bash
git-dashboard
```

## Project Structure

```
tools/git-dashboard/
├── package.json
├── tsconfig.json
├── README.md
├── dist/                # Compiled output (generated by tsc)
└── src/
    ├── index.ts         # CLI entry point (HTTP server + SSE + file watcher)
    ├── types.ts         # Shared TypeScript interfaces
    ├── calendar.ts      # Data processing (commit map, calendar weeks, stats)
    ├── git.ts           # Git data collection (repo name, commits, authors)
    ├── html.ts          # Page template (SVG heatmap, stat cards, layout)
    ├── icons.ts         # SVG icon constants
    └── styles.ts        # CSS theme (dark/light mode)
```
